Everyday Coding--Hello Flask

Overview

The first thing to do when you pick up a new framework is to put together an introductory application to make sure our installation worked. Now, if you're using Test Driven Development as your methodology, you'd first write a test that tests that our application works. But we're not doing that to save some time and effort in learning Python and Flask. You've gone through the process of installing Flask, you have your virtual environment activated, and you're ready to build.
